## Getting Started

NWChem LSP provides intelligent editing support for NWChem input files through the Language Server Protocol (LSP).

### Supported File Extensions

- \`.nw\` - Standard NWChem input files
- \`.nwinp\` - Alternative extension for NWChem input files

## Installation

\`\`\`bash
pip install nwchem-lsp
\`\`\`

## Configuration

### VS Code

Install the NWChem extension from the marketplace or configure manually.

### Neovim

Using nvim-lspconfig.

### Emacs

Using lsp-mode.

## Features

- Syntax Validation
- Auto-completion
- Hover Documentation
- Document Symbols
- Code Actions
- Workspace Symbols
- Semantic Highlighting
- Inlay Hints
