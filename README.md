# Nwchem lsp

Language Server Protocol implementation for nwchem quantum chemistry software.

## Features

- Syntax highlighting
- Auto-completion
- Diagnostics
- Hover documentation

## Installation

```bash
pip install nwchem-lsp
```

## Usage

```bash
nwchem-lsp
```

## Development

```bash
git clone https://github.com/newtontech/nwchem-lsp.git
cd nwchem-lsp
pip install -e ".[dev]"
```
